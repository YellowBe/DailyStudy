## Git 小技巧
+ https://www.zhihu.com/question/20866683/answer/711725573
+ ```git reflog```
+ 这条命令能列出你在 Git 上的所有操作记录，你只要找到 HEAD@{index} 前面所对应的操作索引.
+ ```git reset HEAD@{index}```
+ 回退
